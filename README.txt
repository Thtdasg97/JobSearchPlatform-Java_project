Project Introduction

JobSearchPlatform is a web application designed to connect job seekers with employers. It provides a platform for users to search for job listings, submit applications, and for employers to post job openings.

Functional Description

- User Authentication: Secure account creation and login.
- Job Listings: Easy management of job postings for employers.
- Search and Filters: Seamless job search for seekers based on location, type, and industry.
- CRUD Operations: Full functionality for creating, updating, and deleting job listings.

Deployment Guide (On Local)
Prerequisites
- Java Development Kit (JDK)
- Apache Maven
- MySQL Database

Steps
1. Clone the Repository
git clone https://github.com/Thtdasg97/JobSearchPlatform-Java_project.git
cd JobSearchPlatform-Java_project

2. Configure Database

3. Build and run
mvn clean install
mvn spring-boot:run
Access the platform at http://localhost:8080

4. Platform Access:
Open your browser and go to http://localhost:8080
