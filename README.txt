Title: JobSearchPlatform

JobSearchPlatform is a web application designed to connect job seekers with employers. It provides a platform for users to search for job listings, submit applications, and for employers to post job openings.

Technologies used

Front-end: HTML, CSS, JavaScript
Back-end: Java, Spring Framework, Thymeleaf, JDBC/ Hibernate, MySQL

Responsibilities

Implementing the project following the MVC architectural pattern.
Implementing full CRUD functionality for job listings.
Working with the provided front-end template to implement business logic based on the provided design.
Utilizing additional tools and libraries such as Hibernate, JPA, and Apache Maven for enhanced development.
